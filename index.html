GUIA INSTALACIÓN SERVIDOR WEB Y SERVIDOR VPN EN WINDOWS SERVER

Requisitos previos

¿Que requisitos previos se necesitan para instalar un servidor Web y el servidor VPN en windows server?

Los requisitos necesarios para configurar adecuadamente el servidor son:

  • Sistema Operativo Compatible
  • Hardware Suficiente
  • Conexión a Internet Estable
  • IP Estática o Configuración de DNS
  • Software Antivirus y Antimalware
  • IIS y Componentes Adicionales (opcional)
  • Certificado SSL (Opcional)
  • Configuración de Red.- asignacion de una dirección IP, configurar la puerta de enlace y el servidor DNS según sea necesario.

SERVIDOR WEB (IIS)

IIS, o Internet Information Services, es un conjunto de servicios de servidor web desarrollado por Microsoft para sistemas operativos Windows. Proporciona una plataforma sólida y flexible para alojar, administrar y servir aplicaciones web y servicios.

¿COMO INSTALAR EL SERVIDOR WEB (IIS) EN WINDOWS SERVER?

1. Abre el Administrador del servidor:

Para comenzar, presiona la combinación de teclas Windows + X y selecciona la opción "Administrador del servidor". Luego, haz clic en "Agregar roles y características".

........................

2. Asistente para agregar roles y características:

En la primera pantalla del asistente, simplemente haz clic en "Siguiente" para avanzar al siguiente paso.

........................

3. Tipo de instalación:

Selecciona la opción "Instalación basada en características o roles" y continúa haciendo clic en "Siguiente".

........................

4. Selecciona el servidor:

Asegúrate de que tu servidor esté seleccionado en esta etapa y haz clic en "Siguiente" para avanzar al siguiente paso.

........................

5. Roles de servidor:

En la lista de roles, busca y selecciona "Servicios de servidor web (IIS)".

........................

6. Características requeridas:

Marca la casilla correspondiente y luego haz clic en "Agregar características" para asegurarte de tener todo lo necesario. Luego, continúa con "Siguiente".

........................

7. Roles de servidor:

Una vez completado lo anterior, haz clic en "Siguiente" para avanzar al siguiente paso del asistente.

........................

8. Características:

Continúa con "Siguiente" sin seleccionar características adicionales, ya que las predeterminadas son más que suficientes para agregar una página web sencilla.

........................

9. Rol de servidor web (IIS):

Haz clic en "Siguiente" para avanzar específicamente con la instalación del rol de servidor web (IIS).

........................

10. Servicios de rol:

Antes de proceder, revisa tu selección y, si está correcta, haz clic en "Siguiente".

........................

11. Confirmación:

Confirma la instalación haciendo clic en "Instalar". El asistente llevará a cabo la instalación de IIS y las características relacionadas, este proceso puede tardar unos minutos, así que ten paciencia.

........................

12. Confirma la instalación exitosa:

Una vez completada la instalación, verás un mensaje indicando que la instalación fue exitosa. Haz clic en "Cerrar" para finalizar.

........................

13. Verifica la instalación:

Confirma la presencia de IIS en el panel de administrador del servidor.

........................

14. Verifica la instalación:

Abre un navegador web y escribe la dirección IP de tu servidor o el nombre de dominio que hayas asignado. Si la instalación fue exitosa, deberías visualizar la página predeterminada de IIS.

........................

SERVIDOR VPN

El servicio VPN recurre a la conectividad de Internet y combina tecnologías de tunelización y cifrado de datos para conectarse a clientes y oficinas remotos.

1.-Tener instalado el acceso remoto y las herramientas de DirectAccess, VPN Y Enrutamiento.

........................

2.- después nos dirigimos a enrutamiento y acceso remoto.

........................

3.- Damos click derecho a nuestro servidor a la opción de Configurar y habilitar Enrutamiento y acceso remoto.

........................

4.-le damos a siguiente después le damos a la opción de acceso a red privada virtual (VPN) y NAT.

........................

5.- seleccionamos la opción la cual tenga salida a internet en este caso la WAN.

........................

6.-La asignación de IP se hará de forma automática.

........................

7.-le damos a la opción de ''No, usar Enrutamiento y acceso remoto para autenticar las solicitudes de conexión'' porque no usaremos ningún tipo de servidor RADIUS.

........................

8.-y finalizamos nos saldrá un mensaje le damos a aceptar y con eso instalación del VPN terminaría.
